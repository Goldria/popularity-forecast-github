Популярность проекта

В этом проекте реализовано обучение модели для определения класса популярности проекта на GitHub.

Если требуется определить ранг популярности вашего проекта по всем параметрам репозитория, выберите "Модель классификации".

Если хотите определить возможное количество звезд вашего репозитория по другим параметрам, выберите "Модель регрессии".
